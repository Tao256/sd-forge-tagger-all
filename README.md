# sd-forge-tagger-all

### v1

A web UI plugin based on a modified `WD14-tagger` for `sd-forge` or `forge neo`, which can use tag models such as `WD14`, `Florence2`, etc.

Support 3 types of language for UI interface:`简体中文`、`繁體中文`、`English`,default `English`.

You can change the language and its preset parameters in the webui settings.

Model loads in `"sd-webui\models\tagger_models"`
The output result will be saved in `"sd-webui\output\tagger_output"`

### v1
该webui插件支持`sd-forge` or `forge neo`，可以使用`WD14-tagger`和`Florence2`等模型。

你可以在webui 设置内修改语言及其预设参数，默认`English`语言。

模型放置在 `"sd-webui\models\tagger_models"`
输出结果会保存在 `"sd-webui\output\tagger_output"`
