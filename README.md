WD14 Tagger for sd-webui-forge-neo

This WD14 Tagger is specifically created for sd-webui-forge-neo.

Automatic Model Download

When used for the first time, it will automatically download the models to the \extensions\sd-forge-wd14-tagger-neo\models folder.

(Please wait a moment for the download to complete)
